# Color Palette — AI Email Reply Assistant (2025)

## Brand Colors

The application uses a vibrant, modern 5-color palette designed for excellent readability and visual hierarchy in both light and dark modes.

### Primary Colors

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Pink** | `#EF476F` | Critical actions, alerts, destructive buttons |
| **Gold** | `#FFD166` | Secondary actions, highlights, warm accents |
| **Teal** | `#06D6A0` | Accent, success states, interactive elements |
| **Azure** | `#118AB2` | Primary actions, links, main brand color |
| **Navy** | `#073B4C` | Dark primary, headers, depth |

## Semantic Roles

### Light Mode
- **Primary**: Azure `#118AB2`
- **Secondary**: Gold `#FFD166`
- **Accent**: Teal `#06D6A0`
- **Critical**: Pink `#EF476F`
- **Background**: `#F9FAFB` (light gray)
- **Surface**: `#FFFFFF` (white)
- **Text**: `#1A1A1A` (near black)

### Dark Mode
- **Primary**: Navy `#073B4C`
- **Secondary**: Gold `#E6B85C` (desaturated)
- **Accent**: Teal `#06D6A0`
- **Critical**: Pink `#EF476F`
- **Background**: `#0D1117` (deep dark)
- **Surface**: `#111827` (dark gray)
- **Text**: `#F4F4F4` (off-white)

## Color Contrast

All color combinations meet WCAG AA standards for accessibility:
- Azure on white/light backgrounds: ✓
- Navy on dark backgrounds with white text: ✓
- Pink for critical warnings with white text: ✓
